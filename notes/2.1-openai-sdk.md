The `openai` Node.js SDK is used across the API for chat, embeddings, and summarization.

**Key Files:**

- `api/notes-core.mjs`: `getClient`, `getTextEmbedding`, `getTagEmbedding`, `summarize`
- `api/prompt-core.mjs`: `getClient`, `chatWithTemperatures`, `listModels`

**Usage:**

- A single `getClient(apiKey)` function initializes `new OpenAI({ apiKey })`.
- The client is passed to functions that call `client.chat.completions.create`, `client.embeddings.create`, and `client.models.list`.
- Error handling is managed via `try/catch` blocks in the route handlers.
